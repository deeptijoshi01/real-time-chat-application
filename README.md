# Real-Time Chat Application

A real-time chat application built using HTML, CSS, JavaScript, and WebSockets.
Users can create chat rooms, join existing rooms, see active users, and chat in real time.

## Features
- Real-time messaging using WebSockets
- Create and join chat rooms
- Room switching
- Active users list
- Typing indicator
- WhatsApp-style chat UI
- Dark theme (Peacock color scheme)
- Responsive design
